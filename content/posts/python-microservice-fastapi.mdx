## 为什么选 FastAPI

FastAPI 是 Python 生态中最适合做微服务的框架：

- **性能高**：基于 Starlette + Pydantic，性能接近 Node.js 和 Go
- **自动生成文档**：内置 OpenAPI + Swagger UI，调试接口很方便
- **异步原生**：天然支持 async/await，IO 密集型场景表现优秀
- **类型安全**：Pydantic 做请求校验，减少运行时错误

## 快速搭建

```bash
pip install "fastapi[standard]" uvicorn
```

### 基础应用

```python
from fastapi import FastAPI

app = FastAPI(title="User Service", version="1.0.0")

@app.get("/health")
async def health():
    return {"status": "ok"}
```

```bash
uvicorn app.main:app --reload --port 8080
# 打开 http://localhost:8080/docs 自动生成 API 文档
```

### Pydantic 请求校验

```python
from pydantic import BaseModel, EmailStr

class CreateUserRequest(BaseModel):
    name: str = Field(min_length=2, max_length=50)
    email: EmailStr
    age: int = Field(ge=0, le=150)

@app.post("/users")
async def create_user(req: CreateUserRequest):
    return {"id": 1, **req.model_dump()}
```

参数校验失败时 FastAPI 会自动返回 422 和详细的错误信息。

## 服务注册与发现（Python 版）

Python 微服务同样需要注册中心，Consul 是一个不错的选择：

```python
import consul

c = consul.Consul(host="localhost")

# 服务注册
c.agent.service.register(
    name="user-service",
    service_id="user-service-1",
    address="127.0.0.1",
    port=8080,
    check=consul.Check.http("http://localhost:8080/health", interval="10s"),
)
```

## Docker 部署

```dockerfile
FROM python:3.12-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .

CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8080"]
```

```yml
# docker-compose.yml
version: "3.8"
services:
  user-service:
    build: .
    ports:
      - "8080:8080"
    environment:
      - DATABASE_URL=postgresql://postgres:postgres@db:5432/users

  db:
    image: postgres:16
    environment:
      POSTGRES_DB: users
      POSTGRES_PASSWORD: postgres
```

## gRPC 通信

当服务间需要高性能通信时，Python 通过 grpcio 支持 gRPC：

```protobuf
// user.proto
service UserService {
    rpc GetUser (GetUserRequest) returns (User) {}
}

message GetUserRequest {
    int64 user_id = 1;
}

message User {
    int64 id = 1;
    string name = 2;
    string email = 3;
}
```

```python
import grpc
import user_pb2_grpc, user_pb2

channel = grpc.insecure_channel("localhost:50051")
stub = user_pb2_grpc.UserServiceStub(channel)
user = stub.GetUser(user_pb2.GetUserRequest(user_id=1))
```

## 总结

Python 做微服务的关键是**用好异步、类型系统和容器化**。FastAPI 在这三方面都做得不错。如果你的团队 Python 技术栈更成熟，或者业务偏 AI/数据处理，FastAPI 是一个值得认真考虑的微服务方案。
